## Setup
Use Docker
```bash
  $ docker compose up -d
  $ docker compose exec node bash
  # in a docker container
  $ yarn install
  $ yarn dev
```

Use Node.js
```bash
  # if yarn is not installed
  $ npm install -g yarn
  # install all dependencies in node_modules
  $ yarn install

  $ yarn dev
  # if use commandprompt
  $ yarn dev:windows
```

access http://localhost:8080
